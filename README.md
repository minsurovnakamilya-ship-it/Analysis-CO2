Разработка системы для анализа и прогнозирования выбросов CO₂ с использованием AutoGluon TimeSeries и визуализацией в Streamlit.

Система для:
прогнозирования выбросов CO₂ по странам
анализа влияния экономических факторов (GDP, population, energy)
проведения аналитики временных рядов
визуализации данных на интерактивной карте мира


Технологии:
Python 
Streamlit — веб-интерфейс
AutoGluon TimeSeries — прогнозирование временных рядов (AutoML)
Pandas — обработка данных
Matplotlib / Seaborn — аналитическая визуализация
Plotly — интерактивные графики и карта


Структура проекта:
app.py                # Streamlit
src/
 ├── data_loader.py   # Подготовка и преобразование данных
 ├── model.py         # Обучение и прогнозирование AutoGluon
 └── visualize.py     # Построение графиков прогноза
analysis.ipynb        # Исследовательский анализ данных EDA
requirements.txt      # Зависимости проекта



1. Базовый прогноз
прогноз CO₂ по историческим данным
автоматический выбор модели (AutoML)
2. Прогноз с признаками
учитываются:
GDP (ВВП страны)
Population
Energy consumption (энергопотребление )
многомерное прогнозирование
3. Аналитика
временные ряды
корреляция признаков
влияние факторов
топ стран по выбросам
4. Карта мира
визуализация CO₂ по странам
интерактивная географическая карта
5. Сравнение моделей (ARIMA, ETS, Theta, DeepAR, Transformer, Naive/ SeasonalNaive, AutoGluon Ensemble)
leaderboard AutoGluon
метрики (MASE, RMSE)
анализ качества моделей


Система автоматически обучает несколько моделей (ARIMA, ETS, нейросети) и выбирает лучшую.
